阿里云ECS搭建The Forest游戏服务器

2020年03月19日 1.8k 字 大概 8 分钟 阅读数 次

需自购云服务器或准备一台没用的老电脑当服务器

（无图警告，其实是真的懒得截图了）

准备内容

1.服务器一只（大于等于双核4G内存的云服务器或者老电脑都行），建议选用没有cpu限制的,我的阿里云ecs就有限制，老难受了
2.steamcmd软件
3.一双手

温馨提醒

-本教程为steam平台正版服务器，其他服务器不一样不能联机
-以下主要讲以windows系统的教程，服务器端请给你的服务器用windows server（自用windows server 2019数据中心版用linux也行，但要装wine，操作较麻烦，而且对小白不太友好，我就不详细介绍了），老电脑就用windows就行
-看下面教程有些不知道点哪里的自行百度就知道了
重点：windows系统一定要64位！！！！！

详细教程

1.云服务器首先自己买好，然后弄好实例，这里不详细介绍，阿里云什么的都有这种基本教程
2.进阿里云控制台，找到云服务器实例，点远程连接，记得选用VNC模式（不要用workbench），等待进入系统
3.安装steamcmd，百度一搜就搜的到，下载下来安装运行steamcmd.sh
4.安装完成后，进入steamcmd，这时候应该显示steam>,所以我们输入login 用户名 密码
5.输入完后按回车，他要求你输入验证码，验证码会发到你的邮箱里，直接输入验证码然后回车，出现Logged in OK Waiting for user info…OK表示登录成功
6.设置游戏目录，输入force_install_dir “D:\the forest” #这个目录随你便
7.安装游戏，输入app_update 556450 validate（此处556450是森林这个游戏的服务器版本的ID）
8.等待游戏安装完成，输入quit即可退出steamcmd
9.找到你的安装目录，点开TheForestDedicatedServer.exe，他就是黑屏的不要慌，等待大概10秒就可以关闭它了，这一步目的是为了创建相应的文件
10.在你的云服务器中为TCP打开8766, 27015 和 27016端口，不会的百度，如果是老电脑就为你的路由器打开这三个端口
11.先打开显示隐藏文件（不会记得百度），找到C：\ Users \你的电脑账户名称\ AppData \ LocalLow \ SKS \ TheForestDedicatedServer \ ds文件夹，就会发现Server.cfg配置文件，然后打开它
12.里面大多数不用改，具体怎么调整请往下看注释2，先讲建议或必须修改的东西：
（注意：“//”符号后面的都是注释，不懂干嘛的可以仔细看看，没有“//”符号的那行是可以自己修改的，格式为例如：serverIP空格192.168.2.。。。等等，有些只能填off或On,空格千万不要忘）

13.各种东西修改完保存关闭，重新打开TheForestDedicatedServer.exe，等待右下角出现血量饥饿度等东西就代表你服务器开启成功了，请仔细看显示出来的日志有没有问题

注释

注释1

你需要在 https://steamcommunity.com/dev/managegameservers 中为你的服务器创建一个账号。游戏The Forest 的appid是242760,然后复制你的登录令牌到配置文件Server.cfg的serverSteamAccount后面，否则打开TheForestDedicatedServer.exe会提示无法连接到steam

注释2

以下为配置文档几乎所有信息，部分我也不知道干嘛用的，只能你自己理解了

进入服务器游玩方法

1.进入森林the forest游戏
2.多人游戏-联机游戏-专用（互联网）
3.左下角搜索你服务器名字，搜到了就是ok,没搜到就是你服务器没开启成功，如果继续玩存档就点继续，输入密码，再点继续，进去就可以玩了

游戏注意

1.服务器会自动保存，但你自己角色信息要自己找房子保存
2.服务器整个的存档位置位于你服务器C：\Users\你的电脑账户名称\AppData\LocalLow\SKS\TheForestDedicatedServer\ds\Multiplayer\shot1（或者2/3/4/5）下面全是，可以复制到其他电脑上用
3.每个人自己的信息存档位于你自己本地电脑C:\Users\你的电脑账户名称\AppData\LocalLow\SKS\TheForest\一窜数字\Multiplayer\cs下的那一个文件，也可以复制到其他人那里用
4.如注意2和3所说的shot1（或者2/3/4/5）文件夹是主机存档，其文件夹下有个叫guid的文件，记事本打开显示的那窜数字就是你cs文件夹下个人存档的文件名，换不同服务器的时候注意下，数串一样使才能识别出来
